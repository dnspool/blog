linux命令之screen

  • linux
  • tools

做为苦逼SA，有很多的远程执行命令工作，命令执行时长是会影响到执行结果的。这里说的结果是完成或者夭折，你一定懂得… …

会话操作

一个会话可以开出多个窗口。会话有几个状态，分别是attache（连接）、detache（断开）、dead（会话被人为kill掉）、remove（screen -wipe清除的）。

开启会话

screen -S 会话名称 # 指定名称切换会比较方便screen 会话名称就可以了

会话置后台

Ctrl-A d ，将当前会话（可能包含多个窗口）放后台

查看会话

screen -ls

强置后台

对于已经Attached的会话，以便接管要先执行screen -d 会话名称

接管会话

screen -r 会话名称

强制接管

screen -d -r 会话名称

关闭会话

在需要关闭的会话执行exit即可

试图恢复会话，找不到新建

screen -R 会话名称

屏幕切分：

  • 水平切分: C-a S
  • 垂直切分: C-a |
  • 区块互切: C-a Tab
  • 关闭当前区块: C-a X
  • 关闭其它区块: C-a Q

会话共享

screen -x -r 会话名称

窗口操作

在会话中所有命令都以Ctrl+A（以下简写为'C-a'）开始

开启新窗口: C-a c

窗口命名: C-a A 窗口名称

保存窗口: C-a d

切下一窗口: C-a n

切上一窗口: C-a p

互切两窗口: C-a C-a

编号切窗口: C-a 0..9

列出窗口: C-a w

关闭窗口: C-a K(大写)

移动至行首: C-a a， 作用在bash下

时间和负载: C-a t

copy mode: C-a [， 像使用vi一样，可以回滚，搜索，复制

向上翻页: C-b Backward

向下翻页: C-b Forward

光标至左上: H

光标至左下: L

移动至行首: 0

移动至行尾: $

每字符前移: w

每字符后移: b

块区标记: Space （首次标记起点，二次标记终点）

结束copymode: Esc

粘贴以上选定内容: C-a ]

显示帮助: C-a ?
